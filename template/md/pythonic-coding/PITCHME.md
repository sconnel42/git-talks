### Explicit Coding

@snap[west]
Bad:
@snapend
```
def make_complex(*args):
    x, y = args
    return dict(**locals())
```

@snap[west]
Good:
@snapend
```
def make_complex(x, y):
    return {'x': x, 'y': y}
```
